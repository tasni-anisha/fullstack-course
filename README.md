# Student Task Manager (MERN)

## Name
Anisha Tasnim

## Project Description
This is a simple MERN stack project where users can add and delete tasks.

## Technologies Used
- MongoDB
- Express.js
- React
- Node.js

## How to Run

### Backend
cd backend  
node server.js  

### Frontend
cd frontend  
npm start
